ERROR 2: habitaciones.slice is not a function
──────────────────────────────────────────────

PROBLEMA:
└─ CarruselHabitaciones intentaba hacer .slice() en habitaciones
└─ Si habitaciones no era un array (undefined, null, object) → Error crítico
└─ Crash en HomeScreen cuando se renderiza el carrusel

SOLUCIÓN APLICADA:
├─ Validar que habitaciones sea un array antes de slice()
├─ Usar Array.isArray() para validación
├─ Fallback a array vacío si no es válido

CÓDIGO ANTES:
```javascript
const CarruselHabitaciones = ({
  habitaciones = [],
  ...
}) => {
  const habitacionesLimitadas = habitaciones.slice(0, maxCards);
```

CÓDIGO DESPUÉS:
```javascript
const CarruselHabitaciones = ({
  habitaciones = [],
  ...
}) => {
  const habitacionesValidas = Array.isArray(habitaciones) ? habitaciones : [];
  const habitacionesLimitadas = habitacionesValidas.slice(0, maxCards);
```

ARCHIVO MODIFICADO:
└─ CarruselHabitaciones.js
