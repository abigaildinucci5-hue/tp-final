## 🔄 FLUJO DE AUTENTICACIÓN OAUTH

### **Google OAuth:**
```
1. Usuario click en "Continuar con Google"
2. Se abre navegador/modal con login de Google
3. Usuario autoriza la aplicación
4. Google redirige con código de autorización
5. Backend intercambia código por access_token
6. Backend busca/crea usuario en la BD
7. Backend genera JWT propio
8. Frontend guarda JWT y redirige
```

### **GitHub OAuth:**
```
1. Usuario click en "Continuar con GitHub"
2. Se abre navegador/modal con login de GitHub
3. Usuario autoriza la aplicación
4. GitHub redirige con código de autorización
5. Backend intercambia código por access_token
6. Backend busca/crea usuario en la BD
7. Backend genera JWT propio
8. Frontend guarda JWT y redirige
